### Run Locally

To run **Transportio** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/adnan-bhaldar/transportation.git
```

Windows:

```bash
git clone https://github.com/adnan-bhaldar/transportation.git
```
